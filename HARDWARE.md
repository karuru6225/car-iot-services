# ハードウェア設計メモ

## 概要

12V 車載バッテリー（サブバッテリー）から電源を取り、2系統のバッテリー電圧を監視する基板。
**案B（ESP32-S3-MINI-1 基板直付け）で設計確定。**

| 案 | MCU | 接続方式 | 状態 |
| -- | --- | -------- | ---- |
| 案A（従来） | M5Atom S3（外付け） | Grove ケーブル 1 本（5V / GND / SDA / SCL） | 廃案 |
| **案B** | ESP32-S3-MINI-1-N8（基板直付け） | カステレーションはんだ付け | **採用** |

---

## システム構成

### 案A（M5Atom S3 外付け）

```text
[12V サブバッテリー]
        ↓
[本基板]
  保護回路（F1 → TVS1 → D1）
        ↓
  LM2596（12V → 5V）
        ↓
  CN4-A: GROVE コネクタ（5V + GND + SDA + SCL）
        ↓
[M5Atom S3] ← Grove ケーブル 1 本
  └── UART（GPIO5/6）→ U128（SIM7080G）← 別途 Grove 4線接続

[メインバッテリー +] → 分圧回路 → ADS1115 AIN0/AIN1
[サブバッテリー +]  → 分圧回路 → ADS1115 AIN2/AIN3
```

### 案B（ESP32-S3-MINI-1 基板直付け）

```text
[12V サブバッテリー]
        ↓
[本基板]
  保護回路（F1 → TVS1 → D1）
        ↓
  LM2596（12V → 5V）
        ↓
  ESP32-S3-MINI-1-N8（基板直付け）
    I2C ──→ ADS1115
    UART ──→ CN4-B: Grove コネクタ → U128（SIM7080G）

[メインバッテリー +] → 分圧回路 → ADS1115 AIN0/AIN1
[サブバッテリー +]  → 分圧回路 → ADS1115 AIN2/AIN3

[CN5: リレー出力] ←── FET × 3（GPIO 制御）※案B のみ
```

## 分圧回路

### 定数

| 定数 | 値 |
| ---- | -- |
| R1（上側） | 680kΩ 1% |
| R2（下側） | 11kΩ 1% |
| 分圧比 | 11k / (680k + 11k) ≒ 1/62.8 |
| ADS1115 ゲイン | GAIN_EIGHT（±0.512V） |
| 理論分解能 | 約 1mV / LSB（バッテリー端換算） |

### 対応電圧範囲

- 最大: 約 32V（ADS1115 入力 0.51V）
- LiFePO4 4S: 10.0V（放電）～ 14.6V（充電）→ 0.159V ～ 0.233V
- 鉛バッテリー 12V: 10.5V ～ 14.7V → 0.167V ～ 0.234V

どちらも ±0.512V レンジ内に収まる。

## PCB レイアウト指針

### ゾーン分離

```text
┌──────────────────────────────────────────────────┐
│ [CN1: 12V入力]                                   │
│      ↓                                           │
│ F1 → TVS1 → D1                                  │
│                                                  │
│ ┌─[スイッチング領域]──┐    ┌─[アナログ領域]──────┐ │
│ │ LM2596             │    │ ADS1115            │ │
│ │ L1（スイッチングノード近傍）│ │ C7/C8（VDD直近）  │ │
│ │ D2                 │    │ R5/R6（プルアップ） │ │
│ │ C1 / C2            │    │                    │ │
│ └────────────────────┘    └────────────────────┘ │
│           ↑ GND は 1 点で結合                     │
│                                                  │
│ [CN2: BAT1] → R1/R2 → ADS1115 AIN0/1            │
│ [CN3: BAT2] → R3/R4 → ADS1115 AIN2/3            │
│                                                  │
│ [CN4-A: Grove → M5Atom S3]  ※案A               │
│ [ESP32-S3-MINI-1 + CN4-B: Grove → U128]  ※案B  │
└──────────────────────────────────────────────────┘
```

### 重要ルール

1. **LM2596 のスイッチングノード**（L1・D2 が接続されるノード）のパターンは短く・面積を小さく保つ。最もノイズを放射する箇所。
2. ADS1115 は LM2596 から**基板の反対側**に配置する。
3. 分圧抵抗（R1～R4）は ADS1115 の近くに置き、パターンを短くする。
4. C7/C8（デカップリング）は ADS1115 の VDD ピン直近に配置する。
5. GND プレーン（ベタ）を使う場合も、スイッチング領域とアナログ領域の GND を **1 点で接続**する（スター GND）。
6. 案B: ESP32-S3-MINI-1 の**全 GND ピンは基板 GND に接続**する（放熱・ノイズ対策）。

## 精度向上のための推奨事項

- 分圧抵抗は **1% 精度品**を使用（5% 品は不可）
- 同一温度ドリフト品の抵抗を近傍配置してペア動作させる
- 必要に応じてソフトウェアでキャリブレーション補正を追加する
- 高精度が必要な場合は変換レートを 8SPS に設定し複数サンプル平均化を行う

## 回路検証結果（m5atom_power_adc.kicad_sch）

### リレー駆動トランジスタ（Q3/Q4/Q7）

MMBT2222A、`(mirror y)` 配置。ピン位置を座標計算で検証済み。

- **E（pin2） → GND2**（コモンエミッタ確認）
- **C（pin3） → リレーコイル低電位側 + LED陰極**
- LED点灯電流：I = (12 - Vce_sat - Vf) / 5.1kΩ ≈ **1.9mA**（正常点灯）

> kicad-happy / kicad-netlist-tool の解析ツールは `mirror` + `angle` 組み合わせ時に
> E/C を誤判定するバグがある。実回路はコモンエミッタで正しい。

### RelayXSense 回路（R27-R32）

| 状態 | V_sense | 論理 |
| ---- | ------- | ---- |
| リレーOFF（コイル接続時） | 約 2.55V | HIGH |
| リレーON（Q または手動SW） | 約 0V | LOW |

**負論理**（HIGH=OFF）。コイル未接続時も 0V になる点に注意。

### USB-C SBU1/SBU2 ピン

USB 2.0 用途のみのため **NC（未接続）** が正しい。

## ADS1115 チャンネル割り当て

| チャンネル | 用途 | 測定方式 | ゲイン |
| ---------- | ---- | -------- | ------ |
| AIN0/AIN1 差動 | サブバッテリー電圧（J5） | 差動 | GAIN_EIGHT (±0.512V) |
| AIN2/AIN3 差動 | 車メインバッテリー電圧（J6） | 差動 | GAIN_EIGHT (±0.512V) |

分圧比: 22k / (680k + 22k) ≈ 0.0313。12V → 376mV、15V → 470mV（FSR内）。

複数チャンネル読み取り時は**チャンネル切り替え後に変換完了を待ってから読む**こと（`readADC_Differential_X_Y()` はシングルショットのため自動待機）。

## 拡張予定回路

### サブバッテリー電流計測（INA226）

| 項目 | 内容 |
| ---- | ---- |
| IC | INA226（I2C、アドレス 0x40） |
| 接続 | 既存 SDA/SCL に並列追加（I2C プルアップ追加不要） |
| シャント | ローサイド（Battery− と GNDA の間）、1mΩ バスバー型 |
| 想定電流 | 最大 80A（常用 40A） |
| 読み取り | 電流・電圧・電力レジスタ（DeepSleep 中の積算は不可） |

```text
GND_BUS ─── INA226(IN+)
INA226(IN−) ─── R_SHUNT(1mΩ) ─── Battery(−)/GNDA
```

### バッテリー残量積算（LTC2944、将来検討）

DeepSleep 中も積算を継続したい場合に使用。**ハイサイド**接続専用 IC。

| 項目 | 内容 |
| ---- | ---- |
| シャント | 0.5mΩ（±50mV / 80A） |
| 接続 | ハイサイド（Battery+ と負荷の間） |
| 特徴 | 内蔵 16bit クーロンカウンタ、電圧・温度も計測 |

INA226 と排他選択。正確な残量記録が必要な場合は LTC2944 を優先する。

### サブ → メインバッテリー緊急充電回路

メインバッテリー電圧低下時にサブから一時的に補充する回路。

**回路構成（AO3401A + MMBT2222A 流用）：**

```text
Sub(+) ─── AO3401A(P-ch) ─── Schottkyダイオード ─── R_limit ─── Main(+)
               ↑ MMBT2222Aレベルシフタ経由
           ESP32 GPIO
```

| 部品 | 値 | 備考 |
| ---- | -- | ---- |
| Q（Pch MOSFET） | AO3401A | 既存 BOM 流用 |
| レベルシフタ | MMBT2222A | 既存 BOM 流用 |
| Schottky | 逆流防止用 | Vf ≈ 0.3V |
| R_limit | **1Ω 合成**（1Ω 5W × 4個、2直2並） | 合成 20W |

**制御ロジック：**

```
開始: V_main < 11.5V かつ V_sub > 13.0V
停止: V_main > 12.3V または 経過時間 > 20分（安全タイマー）
間欠動作: 電圧差に応じて ON 時間を変化
  差 < 0.5V → 60秒 ON
  差 < 1.0V → 30秒 ON
  差 < 2.0V → 10秒 ON
  差 ≥ 2.0V →  3秒 ON
```

最大電流: (14 - 11 - 0.3) / 1Ω = **2.7A**、発熱: 2.7² × 1 = **7.3W**（間欠）。

### CAN Bus（将来検討）

ESP32-S3 内蔵 TWAI（CAN 2.0B）を使用。外付けトランシーバーのみ追加。

| 部品 | 内容 |
