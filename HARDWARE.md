# ハードウェア設計メモ

## 概要

12V車載バッテリー（サブバッテリー）から電源を取り、2系統のバッテリー電圧を監視する基板。
M5Atom S3 への給電と ADS1115 との I2C 通信を GROVE ケーブル 1 本で束ねる。

---

## システム構成

```
[12V サブバッテリー]
        ↓
[本基板]
  保護回路（F1 → TVS1 → D1）
        ↓
  LM2596（12V → 5V）
        ↓
  GROVE コネクタ（5V + GND + SDA + SCL）
        ↓
[M5Atom S3] ← GROVE ケーブル 1 本

[メインバッテリー +] → 分圧回路 → ADS1115 AIN0/AIN1
[サブバッテリー +]  → 分圧回路 → ADS1115 AIN2/AIN3
```

## 分圧回路

### 定数

| 定数 | 値     |
| ---- | ------ |
| R1（上側） | 680kΩ 1% |
| R2（下側） | 11kΩ 1%  |
| 分圧比 | 11k / (680k + 11k) ≒ 1/62.8 |
| ADS1115 ゲイン | GAIN_EIGHT（±0.512V） |
| 理論分解能 | 約 1mV / LSB（バッテリー端換算） |

### 対応電圧範囲

- 最大: 約 32V（ADS1115 入力 0.51V）
- LiFePO4 4S: 10.0V（放電）～ 14.6V（充電）→ 0.159V ～ 0.233V
- 鉛バッテリー 12V: 10.5V ～ 14.7V → 0.167V ～ 0.234V

どちらも ±0.512V レンジ内に収まる。

## GROVE コネクタ（CN4）ピン配置

M5Atom S3 の Port A（HY2.0-4P）に直結する。

| ピン | 信号 | 備考                    |
| ---- | ---- | ----------------------- |
| 1    | GND  |                         |
| 2    | VCC  | 5V（LM2596 出力）       |
| 3    | SDA  | ADS1115 SDA（G38）      |
| 4    | SCL  | ADS1115 SCL（G39）      |

Port A の VCC は 5V 出力であることを実機で確認済み。

## PCB レイアウト指針

### ゾーン分離

```
┌──────────────────────────────────────────────────┐
│ [CN1: 12V入力]                                   │
│      ↓                                           │
│ F1 → TVS1 → D1                                  │
│                                                  │
│ ┌─[スイッチング領域]──┐    ┌─[アナログ領域]──────┐ │
│ │ LM2596             │    │ ADS1115            │ │
│ │ L1（スイッチングノード近傍）│ │ C7/C8（VDD直近）  │ │
│ │ D2                 │    │ R5/R6（プルアップ） │ │
│ │ C1 / C2            │    │                    │ │
│ └────────────────────┘    └────────────────────┘ │
│           ↑ GND は 1 点で結合                     │
│                                                  │
│ [CN2: BAT1] → R1/R2 → ADS1115 AIN0/1            │
│ [CN3: BAT2] → R3/R4 → ADS1115 AIN2/3            │
│                                                  │
│ [CN4: GROVE] ← 5V + SDA + SCL + GND             │
└──────────────────────────────────────────────────┘
```

### 重要ルール

1. **LM2596 のスイッチングノード**（L1・D2 が接続されるノード）のパターンは短く・面積を小さく保つ。最もノイズを放射する箇所。
2. ADS1115 は LM2596 から**基板の反対側**に配置する。
3. 分圧抵抗（R1～R4）は ADS1115 の近くに置き、パターンを短くする。
4. C7/C8（デカップリング）は ADS1115 の VDD ピン直近に配置する。
5. GND プレーン（ベタ）を使う場合も、スイッチング領域とアナログ領域の GND を **1 点で接続**する（スター GND）。

## ファームウェアとの対応

| 設定            | 値                              |
| --------------- | ------------------------------- |
| I2C アドレス    | `0x48`（ADDR → GND）            |
| SDA ピン        | G38                             |
| SCL ピン        | G39                             |
| ゲイン          | `GAIN_EIGHT`（±0.512V）         |
| 差動入力 ch1    | `readADC_Differential_0_1()`    |
| 差動入力 ch2    | `readADC_Differential_2_3()`    |

コード変更時は `ads.begin(0x48)` に変更し、2 系統の読み取り関数を追加する。

---

## 精度向上のための推奨事項

- 分圧抵抗は **1% 精度品**を使用（5% 品は不可）
- 同一温度ドリフト品の抵抗を近傍配置してペア動作させる
- 必要に応じてソフトウェアでキャリブレーション補正を追加する
- 高精度が必要な場合は変換レートを 8SPS に設定し複数サンプル平均化を行う
