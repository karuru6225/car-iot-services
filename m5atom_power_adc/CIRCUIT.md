## 電源回路

### 保護回路

| 部品 | 値 | 動作 |
| ---- | -- | ---- |
| F1 | ポリヒューズ 1.5A | 過電流で抵抗増大、自己復帰 |
| D1 | SMAJ16A TVS | クランプ電圧 18.2V（typ）、車サージ吸収。逆接保護なし |

> **変更点**: 旧設計の TVS1（P6KE18A）＋ D1（SB360 逆接保護）から SMAJ16A（TVS のみ）に変更。逆接保護ダイオードは削除。

### LM2596S-5（12V → 5V）

固定 5V 出力品（FB ピンを出力に直結）。

| 定数 | 値 | 備考 |
| ---- | -- | ---- |
| VIN | 12V（8〜40V 動作） | |
| VOUT | 5V | 固定出力品 |
| 動作周波数 | 150kHz | 内部固定 |
| L1 | 68μH / 1.5A | Würth PSPMAA0605H-680M-IGF |
| D2 | SS34 ショットキー 3A/40V | フリーホイールダイオード |
| C1 | 220μF / 35V 電解 | 入力平滑 |
| C2 | 220μF / 10V 電解 | 出力平滑 |
| R15 | 10kΩ | ON/OFF ピン → +12V プルアップ（常時 ON） |
| FB1 | 500mA / 600Ω@100MHz フェライトビーズ | +5V バスとアナログ電源（U5 入力）の分離 |

**設計計算：**

```text
デューティ比:  D = VOUT / VIN = 5/12 ≒ 0.42（42%）
インダクタ電流リップル: ΔIL = (VIN − VOUT) × D / (f × L)
  = (12 − 5) × 0.42 / (150kHz × 68μH) ≒ 0.29A
ピーク電流: Iout(max) + ΔIL/2 ≒ 0.5A + 0.145 ≒ 0.65A（< L1 定格 1.5A ✓）
効率: 約 80%（typ）
```

### USB-C 受電（OR 接続）

| 部品 | 値 | 備考 |
| ---- | -- | ---- |
| J1 | USB_C_Receptacle_USB2.0_14P | USB 2.0、SBU1/SBU2 = NC |
| R2, R3 | 5.1kΩ | CC1/CC2 → GND（UFP = 受電側 識別） |
| D3 | LESD5D5.0CT1G | VBUS-GND 間 TVS 保護 |
| D5, D6 | LESD5D5.0CT1G | USB D−/D+ ESD 保護 |
| D7 | SSL36 ショットキー | VBUS → +5V バスへの OR ダイオード、逆流防止 |

LM2596 出力と USB-C VBUS をショットキー D7 で OR 接続。電圧の高い方が優先的に +5V バスを供給する。

### AMS1117-3.3 × 2

ESP32-S3-MINI-1 へ 3.3V を供給。

| 定数 | 値 | 備考 |
| ---- | -- | ---- |
| VIN | +5V | |
| VOUT | +3.3V | |
| C3 | 100nF | 入力パスコン |

## 電圧測定回路（ADS1115）

ADS1115 は +3.3VA で駆動。I2C プルアップ（R6/R7/R14）も +3.3VA に接続。

### 分圧回路

サブバッテリー（BATT0）とメインバッテリー（BATT1）を各 2 ピンコネクタで接続。バッテリー両端の電圧差を差動測定。

| 定数 | 値 | 備考 |
| ---- | -- | ---- |
| R_upper (R8, R4) | 680kΩ 1% | BATT+側の直列分圧抵抗 |
| R_lower (R9, R5) | 22kΩ 1% | BATT−側の直列抵抗（GND へ） |
| 分圧比 | 22/(680+22) = 0.03134 | |
| ADS1115 ゲイン | GAIN_EIGHT（±0.512V） | |
| 分解能 | 15.625μV/LSB | 16bit、±0.512V 範囲 |
| バッテリー端換算 | 0.498mV/LSB | |

**電圧変換式：**

```text
V_battery = ADC_raw × (0.512 / 32768) / (22 / 702) × 1000  [mV]
           = ADC_raw × 0.498  [mV]
```

**測定範囲：**

| 電池種別 | 電圧範囲 | ADS1115 入力 |
| -------- | -------- | ------------ |
| 鉛 12V | 10.5〜14.7V | 0.329〜0.461V |
| LiFePO4 4S | 10.0〜14.6V | 0.313〜0.457V |
| 最大（FSR） | 16.3V | 0.512V |

### AIN 入力保護・フィルタ

各 AIN ピンに直列保護抵抗（100Ω）と RC アンチエイリアシングフィルタを追加。

| 部品 | 接続先 | 備考 |
| ---- | ------ | ---- |
| R13（100Ω） | BATT0_P → AIN0 | 直列保護 |
| R12（100Ω） | BATT0_N → AIN1 | 直列保護 |
| R11（100Ω） | BATT1_P → AIN2 | 直列保護 |
| R10（100Ω） | BATT1_N → AIN3 | 直列保護 |
| C10（220nF C0G） | AIN0-AIN1 差動 | 差動ノイズ除去、fc ≈ 7.2kHz |
| C13（220nF C0G） | AIN2-AIN3 差動 | 差動ノイズ除去、fc ≈ 7.2kHz |
| C11, C12（22nF C0G） | AIN1, AIN0 → GNDA | コモンモード除去、fc ≈ 72kHz |
| C14, C15（22nF C0G） | AIN3, AIN2 → GNDA | コモンモード除去、fc ≈ 72kHz |

### ADS1115 設定

| 項目 | 値 |
| ---- | -- |
| I2C アドレス | 0x48（ADDR → GND） |
| ゲイン | GAIN_EIGHT（±0.512V） |
| チャンネル ch0 | AIN0/AIN1 差動（サブバッテリー J5） |
| チャンネル ch1 | AIN2/AIN3 差動（メインバッテリー J6） |
| VDD | +3.3VA（U5 出力） |
| R6, R7 | 10kΩ SDA/SCL プルアップ → +3.3VA |
| R14 | 100kΩ ALERT/RDY プルアップ → +3.3VA |
| ADC_RDY | ESP32 GPIO へ接続（変換完了割り込み） |

## リレー制御回路

### NPN BJT ドライバ（Q3/Q4/Q7）

外部リレーコイル（12V / 46mA 以上）を GPIO で駆動。3 チャンネル共通構成。

| 部品 | 値 | 備考 |
| ---- | -- | ---- |
| Q3/Q4/Q7 | MMBT2222A NPN | hFE ≥ 100、Vceo = 40V、SOT-23 |
| R19/R20/R25（ベース） | 1kΩ | ベース電流 Ib = (3.3−0.7)/1k = 2.6mA |
| D11/D12/D17（フライバック） | 1N4148WS | コイル逆起電力吸収 |
| R23/R24/R26（LED 並列） | 5.1kΩ | リレー LED 電流制限 |

**飽和確認：**

```text
Ib = (3.3 − 0.7) / 1kΩ = 2.6mA
Ic（リレーコイル 46mA）
hFE × Ib = 100 × 2.6 = 260mA >> 46mA → 飽和動作 ✓
```

### チャンネル構成

| ch | BJT | ベース R | フライバック D | LED R | リレー出力 J | 外部 SW J |
| -- | --- | -------- | -------------- | ----- | ----------- | --------- |
| 0 | Q3 | R19 | D11 | R23 | J7 | J11 |
| 1 | Q4 | R20 | D12 | R24 | J8 | J12 |
| 2 | Q7 | R25 | D17 | R26 | J10 | J13 |

外部スイッチ（J11/J12/J13）と ESP32 GPIO の OR 動作：

- J11/J12/J13 に手動スイッチを接続するとコレクタを直接プルダウンできる
- ESP32 GPIO と手動スイッチのどちらかが ON であればリレーが駆動

### リレー LED（D15/D16/D18）

リレーコイルと並列に接続。コレクタに引き込まれることで点灯。

```text
+12V ─── リレーコイル ─── Collector
  └──── R(5.1kΩ) + LED ─── Collector  ← コイルと並列
```

```text
If = (12 − Vce_sat − Vf) / 5.1kΩ
   = (12 − 0.2 − 2.1) / 5100 ≈ 1.9mA
```

LED: Würth 150060YS75000（黄、Vf=2.1V typ）

### リレーコイルセンシング（RelayXSense）

外部スイッチによるリレー ON を検出する分圧回路。コレクタノードを監視。

```text
+12V ─── 100kΩ ─── Vsense(GPIO 入力) ─── 27kΩ ─── GND
```

| ch | R_top | R_bottom | センス信号 |
| -- | ----- | -------- | ---------- |
| 0 | R31（100kΩ） | R32（27kΩ） | Relay0Sense |
| 1 | R29（100kΩ） | R30（27kΩ） | Relay1Sense |
| 2 | R27（100kΩ） | R28（27kΩ） | Relay2Sense |

| 状態 | Vsense | GPIO |
| ---- | ------ | ---- |
| リレー OFF（Collector 開放） | 12 × 27/127 = **2.55V** | HIGH（VIH = 2.475V ✓） |
| リレー ON（Collector ≈ 0V） | **≈ 0V** | LOW |

**負論理**（HIGH = OFF）。コイル未接続時も 0V になる。

## パワースイッチ回路

LTE モジュール（SIM7080G / J2）と Grove 2（J9）への +5V 電源を ESP32 GPIO でスイッチ。2 系統独立。

### 回路構成（AO3401A + MMBT2222A）

```text
+5V ─── AO3401A(S) ── AO3401A(D) ── 負荷
              |
           Gate ── R(100kΩ) ── +5V   ← Gate プルアップ（FET OFF 方向）
              |
           MMBT2222A(C)
           MMBT2222A(B) ── R(1kΩ?) ── ESP32 GPIO（LTE_EN / Res0_EN）
           MMBT2222A(E) → GND
```

| ch | P-MOSFET | NPN | Gate R | 負荷 | GPIO 信号 |
| -- | -------- | --- | ------ | ---- | --------- |
| LTE | Q1 | Q6 | R18（100kΩ） | J2（LTE / SIM7080G） | LTE_EN |
| Grove2 | Q5 | Q2 | R21（100kΩ） | J9（Grove 2） | Res0_EN |

| GPIO | MMBT2222A | AO3401A Gate | FET | 出力 |
| ---- | --------- | ------------ | --- | ---- |
| HIGH | ON | ≈ 0V（Vgs < −Vth） | ON | +5V 出力 |
| LOW | OFF | ≈ +5V（Vgs = 0V） | OFF | 遮断 |

## 暗電流（ディープスリープ時）

ESP32-S3-MINI-1 がディープスリープ、パワースイッチで LTE / Grove を遮断した状態。

| 部位 | 電流 | 供給元 | 備考 |
| ---- | ---- | ------ | ---- |
| LM2596 Iq | **5mA** | 12V バス | 支配的。ディープスリープ時の主消費源 |
| AMS1117-3.3 U3 Iq | ~5mA | 5V バス | |
| AMS1117-3.3 U5 Iq | ~5mA | 5V バス（FB1 経由） | |
| ESP32-S3 ディープスリープ | ~10μA | 3.3V | |
| ADS1115 スタンバイ | ~0.1μA | 3.3VA | |
| 電源 LED D8 | 1.8mA | 5V | 常時点灯 |
| 分圧回路（R8+R9、R4+R5） | ~17μA × 2 = 34μA | 12V バス | 680k+22k = 702kΩ、12V 時 |
| **合計（12V 換算）** | **≈ 13mA** | | |

12V / 100Ah サブバッテリーで連続放置時：100Ah / 13mA ≈ **320 時間（約 13 日）**で放電。

スリープ電流を下げるには LM2596 の代わりに低 Iq DC-DC への変更、または LDO × 2 のシャットダウンが有効。
