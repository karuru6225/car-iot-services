# m5atom_power_adc 回路詳細

KiCad プロジェクト `m5atom_power_adc.kicad_sch` の回路定数・設計根拠をまとめたドキュメント。

---

## ブロック図

```text
[12V サブバッテリー / CN1]
        |
        F1（ポリヒューズ 1.5A）
        |
        TVS1（P6KE18A）── GND  ← サージクランプ
        |
        D1（SB360 逆接保護）
        |
     +12V バス ─────────────────────────────────────────
        |               |              |
   LM2596S-5         リレー制御     バッテリー電圧測定
   （DC-DC）         Q3/Q4/Q7       R1～R4 分圧
        |               |              |
      +5V バス ─────────────────     ADS1115
        |                               |
   [USB-C / CN_USB]                   I2C
   D4（OR ダイオード）                   |
        |                         ESP32-S3-MINI-1
   AMS1117-3.3 (U3)                    |
        |                         UART → CN4-B → U128
      +3.3V                            |
        |                         GPIO → リレー制御
   ESP32-S3-MINI-1                     |
   ADS1115                        GPIO → パワースイッチ
```

## 電源回路

### 保護回路

| 部品 | 値 | 動作 |
| ---- | -- | ---- |
| F1 | ポリヒューズ 1.5A | 過電流で抵抗増大、自己復帰 |
| TVS1 | P6KE18A | クランプ電圧 20V（typ）、車サージ（最大 60V）吸収 |
| D1 | SB360 ショットキー 3A/60V | 逆接保護、Vf ≈ 0.5V |

### LM2596S-5（12V → 5V）

固定 5V 出力品（FB ピンを出力に直結）。

| 定数 | 値 | 備考 |
| ---- | -- | ---- |
| VIN | 12V（8〜40V 動作） | |
| VOUT | 5V | 固定出力品 |
| 動作周波数 | 150kHz | 内部固定 |
| L1 | 68μH / 1.5A | Würth PSPMAA0605H-680M-IGF |
| D2 | SS34 ショットキー 3A/40V | フリーホイールダイオード |
| C1 | 220μF / 35V 電解 | 入力平滑 |
| C2 | 220μF / 10V 電解 | 出力平滑 |
| R15 | 10kΩ | ON/OFF ピン → +12V プルアップ（常時 ON） |

**設計計算：**

```text
デューティ比:  D = VOUT / VIN = 5/12 ≒ 0.42（42%）
インダクタ電流リップル: ΔIL = (VIN − VOUT) × D / (f × L)
  = (12 − 5) × 0.42 / (150kHz × 68μH) ≒ 0.29A
ピーク電流: Iout(max) + ΔIL/2 ≒ 0.5A + 0.145 ≒ 0.65A（< L1 定格 1.5A ✓）
効率: 約 80%（typ）
```

### USB-C 受電（OR 接続）

| 部品 | 値 | 備考 |
| ---- | -- | ---- |
| CN_USB | USB-C レセプタクル | USB 2.0、SBU1/SBU2 = NC |
| R_CC1, R_CC2 | 5.1kΩ | CC1/CC2 → GND（UFP = 受電側 識別） |
| D_ESD | LESD5D5.0CT1G | VBUS-GND 間 TVS + D+/D− 保護、SOT-23 |
| C_ESD | 100nF | TVS と並列（高周波 ESD バイパス） |
| D4 | SSL36 ショットキー | VBUS → +5V バスへの OR ダイオード、逆流防止 |

LM2596 出力と USB-C VBUS をショットキーで OR 接続。電圧の高い方が優先的に +5V バスを供給する。

### AMS1117-3.3（5V → 3.3V）

ESP32-S3-MINI-1 と ADS1115 に 3.3V を供給。

| 定数 | 値 | 備考 |
| ---- | -- | ---- |
| VIN | 5V | |
| VOUT | 3.3V | |
| Iq | 約 5mA | ディープスリープ時の暗電流に注意 |
| C_IN | 100nF | 入力パスコン |
| C_OUT | 10μF | 出力安定化（X7R、25V 耐圧） |

## 電圧測定回路（ADS1115）

### 分圧回路

| 定数 | 値 | 備考 |
| ---- | -- | ---- |
| R_upper (R1, R3) | 680kΩ 1% | |
| R_lower (R2, R4) | 22kΩ 1% | |
| 分圧比 | 22/(680+22) = 0.03134 | |
| ADS1115 ゲイン | GAIN_EIGHT（±0.512V） | |
| 分解能 | 15.625μV/LSB | 16bit、±0.512V 範囲 |
| バッテリー端換算 | 0.498mV/LSB | |

**電圧変換式：**

```text
V_battery = ADC_raw × (0.512 / 32768) / (22 / 702) × 1000  [mV]
           = ADC_raw × 0.498  [mV]
```

**測定範囲：**

| 電池種別 | 電圧範囲 | ADS1115 入力 |
| -------- | -------- | ------------ |
| 鉛 12V | 10.5〜14.7V | 0.329〜0.461V |
| LiFePO4 4S | 10.0〜14.6V | 0.313〜0.457V |
| 最大（FSR） | 16.3V | 0.512V |

高電圧側（R_upper=680kΩ）がハイインピーダンスのため、TVS なしでも ADS1115 入力ピンは保護される（115V 印加まで損傷しない計算）。

### ADS1115 設定

| 項目 | 値 |
| ---- | -- |
| I2C アドレス | 0x48（ADDR → GND） |
| ゲイン | GAIN_EIGHT（±0.512V） |
| チャンネル ch1 | AIN0/AIN1 差動（サブバッテリー J5） |
| チャンネル ch2 | AIN2/AIN3 差動（メインバッテリー J6） |
| VDD | 3.3V（U3 出力） |
| C7 | 100nF（VDD パスコン、ADS1115 直近） |
| C8 | 10μF / 25V X7R（VDD デカップリング） |
| R5, R6 | 10kΩ SDA/SCL プルアップ → 3.3V |
| C5, C6 | 100pF AIN-GND ノイズフィルタ（RC カットオフ ≈ 7kHz） |

**GNDA 分離：** ADS1115 の GND（GNDA）は LM2596 GND とフェライトビーズを介して接続し、スイッチングノイズを低減。

## リレー制御回路

### NPN BJT ドライバ（Q3/Q4/Q7）

外部リレーコイル（12V / 100mA 級）を GPIO で駆動。

| 部品 | 値 | 備考 |
| ---- | -- | ---- |
| Q3/Q4/Q7 | MMBT2222A NPN | hFE ≥ 100、Vceo = 40V、SOT-23 |
| R_base | 1kΩ | ベース電流 Ib = (3.3−0.7)/1k = 2.6mA |
| D（フライバック） | 1N4148 | コイル逆起電力吸収 |
| R_LED（コイル並列） | 5.1kΩ | リレー LED 電流制限 |

**飽和確認：**

```text
Ib = (3.3 − 0.7) / 1kΩ = 2.6mA
Ic(リレーコイル) ≦ 100mA
hFE × Ib = 100 × 2.6 = 260mA >> 100mA → 飽和動作 ✓
```

### リレー LED（D15/D16/D18）

リレーコイルと並列に接続。コレクタに引き込まれることで点灯。

```text
+12V ─── リレーコイル ─── Collector
  └──── R(5.1kΩ) + LED ─── Collector  ← コイルと並列
```

```text
If = (12 − Vce_sat − Vf) / 5.1kΩ
   = (12 − 0.2 − 2.1) / 5100 ≈ 1.9mA
```

LED: Würth 150060YS75000（黄、Vf=2.1V typ）

### リレーコイルセンシング（RelayXSense）

外部スイッチによるリレー ON を検出する分圧回路。コレクタノードを監視。

```text
+12V ─── 100kΩ ─── Vsense(GPIO 入力) ─── 27kΩ ─── GND
```

| 状態 | Vsense | GPIO |
| ---- | ------ | ---- |
| リレー OFF（Collector 開放） | 12 × 27/127 = **2.55V** | HIGH（VIH = 2.475V ✓） |
| リレー ON（Collector ≈ 0V） | **≈ 0V** | LOW |

**負論理**（HIGH = OFF）。コイル未接続時も 0V になる。

## パワースイッチ回路

LTE モジュール（U128）および Grove 予備コネクタへの 5V 電源を ESP32 GPIO でスイッチ。

### 回路構成（AO3401A + MMBT2222A）

```text
+5V ─── AO3401A(S) ── AO3401A(D) ── 負荷
              |
           Gate ── R(10kΩ) ── +5V   ← Gate プルアップ（FET OFF 方向）
              |
           MMBT2222A(C)
           MMBT2222A(B) ── R(1kΩ) ── ESP32 GPIO
           MMBT2222A(E) → GND
```

| GPIO | MMBT2222A | AO3401A Gate | FET | 出力 |
| ---- | --------- | ------------ | --- | ---- |
| HIGH | ON | ≈ 0V（Vgs < −Vth） | ON | 5V 出力 |
| LOW | OFF | ≈ +5V（Vgs = 0V） | OFF | 遮断 |

## 暗電流（ディープスリープ時）

ESP32-S3-MINI-1 がディープスリープ、パワースイッチで LTE / Grove を遮断した状態。

| 部位 | 電流 | 供給元 | 備考 |
| ---- | ---- | ------ | ---- |
| LM2596 Iq | **5mA** | 12V バス | 支配的。ディープスリープ時の主消費源 |
| AMS1117-3.3 Iq | ~5mA | 5V バス | |
| ESP32-S3 ディープスリープ | ~10μA | 3.3V | |
| ADS1115 スタンバイ | ~0.1μA | 3.3V | |
| 電源 LED D8 | 1.8mA | 5V | 常時点灯 |
| 分圧回路（R1～R4） | ~17μA × 2 = 34μA | 12V バス | 680k+22k = 702kΩ、12V 時 |
| **合計（12V 換算）** | **≈ 9mA** | | |

12V / 100Ah サブバッテリーで連続放置時：100Ah / 9mA ≈ **460 時間（約 19 日）**で放電。

スリープ電流を下げるには LM2596 の代わりに低 Iq LDO または 低 Iq DC-DC への変更が有効。
