# ハードウェア設計メモ

## 概要

12V 車載バッテリー（サブバッテリー）から電源を取り、2系統のバッテリー電圧を監視する基板。
**案B（ESP32-S3-MINI-1 基板直付け）で設計確定。**

| 案 | MCU | 接続方式 | 状態 |
| -- | --- | -------- | ---- |
| 案A（従来） | M5Atom S3（外付け） | Grove ケーブル 1 本（5V / GND / SDA / SCL） | 廃案 |
| **案B** | ESP32-S3-MINI-1-N8（基板直付け） | カステレーションはんだ付け | **採用** |

---

## システム構成

### 案A（M5Atom S3 外付け）

```text
[12V サブバッテリー]
        ↓
[本基板]
  保護回路（F1 → TVS1 → D1）
        ↓
  LM2596（12V → 5V）
        ↓
  CN4-A: GROVE コネクタ（5V + GND + SDA + SCL）
        ↓
[M5Atom S3] ← Grove ケーブル 1 本
  └── UART（GPIO5/6）→ U128（SIM7080G）← 別途 Grove 4線接続

[メインバッテリー +] → 分圧回路 → ADS1115 AIN0/AIN1
[サブバッテリー +]  → 分圧回路 → ADS1115 AIN2/AIN3
```

### 案B（ESP32-S3-MINI-1 基板直付け）

```text
[12V サブバッテリー]
        ↓
[本基板]
  保護回路（F1 → TVS1 → D1）
        ↓
  LM2596（12V → 5V）
        ↓
  ESP32-S3-MINI-1-N8（基板直付け）
    I2C ──→ ADS1115
    UART ──→ CN4-B: Grove コネクタ → U128（SIM7080G）

[メインバッテリー +] → 分圧回路 → ADS1115 AIN0/AIN1
[サブバッテリー +]  → 分圧回路 → ADS1115 AIN2/AIN3

[CN5: リレー出力] ←── FET × 3（GPIO 制御）※案B のみ
```

### リレー制御回路（案B のみ）

外部リレー（最大 3 個）を GPIO で制御するための NPN BJT ドライバ回路。

| Ref | 種別 | 仕様 | 数量 | 備考 |
| --- | ---- | ---- | ---- | ---- |
| Q3, Q4, Q7 | NPN BJT | MMBT2222A（SOT-23） | 3 | hFE ≥ 100、Vceo = 40V |
| R（ベース） | チップ抵抗 | 1kΩ | 3 | Ib = (3.3−0.7)/1kΩ = 2.6mA、コイル電流 100mA に対して十分な飽和 |
| D（コイル並列） | 小信号ダイオード | 1N4148 | 3 | フライバックダイオード（リレーコイル並列、逆向き） |

```text
GPIO ──[1kΩ]──→ Base
                MMBT2222A
                Collector ──→ リレーコイル低電位側（コイル高電位側: +12V）
                Emitter  → GND

リレーコイル: +12V ─── コイル ─── Collector
                           │
                        [1N4148 逆向き]（フライバック）
```

### パワースイッチ回路（LTE モジュール・Grove 予備コネクタ電源）

AO3401A（P-ch MOSFET）+ MMBT2222A（NPN レベルシフタ）で 5V ラインを ESP32 GPIO でスイッチする。

```text
+5V ─── AO3401A(S) ─── AO3401A(D) ─── 負荷（LTE / Grove）
           ↑ Gate
         MMBT2222A(C) ── 10kΩ ── +5V
         MMBT2222A(B) ── 1kΩ ── ESP32 GPIO
         MMBT2222A(E) → GND
```

GPIO HIGH → MMBT2222A ON → Gate ≈ 0V（Vgs < −Vth）→ AO3401A ON → 電源供給  
GPIO LOW → MMBT2222A OFF → Gate ≈ +5V（Vgs = 0V）→ AO3401A OFF → 電源遮断

### ブザー回路（案B のみ）

AO3401A（P-ch MOSFET）ハイサイドスイッチで電磁パッシブブザーを駆動。

```text
+3.3V（U5出力） ─── Source(AO3401A/Q10)
                        │
                     Drain ─── LS1(+) ─── LS1(−) ─── GND
                                    │
                                [D19: 1N4148（フライバック）]
                                カソード → LS1(+)/Drain 側
                                アノード → LS1(−)/GND 側

GPIO34(SPEAKER) ──[R39: 10kΩ]──→ Gate(Q10)
```

GPIO LOW → Gate = 0V、Vgs = −3.3V → AO3401A ON → ブザー発音（**負論理**）  
GPIO HIGH → Gate = +3.3V、Vgs = 0V → AO3401A OFF → 消音

| 部品 | 値 | 備考 |
| ---- | -- | ---- |
| Q10 | AO3401A（SOT-23） | 既存 BOM 流用 |
| R39 | 10kΩ | ゲート直列抵抗（プルアップ兼用） |
| D19 | 1N4148（SOD-323 等） | フライバック保護 |
| LS1 | MLT-8530（LCSC: C94599） | 電磁パッシブ、8.5×8.5mm SMD、2.5〜4.5V |
| 電源 | +3.3V（U5 出力） | U3（+3.3VA）には接続しない |
| GPIO | GPIO34 | LEDC PWM 対応 |

**駆動条件（最大音量）：** 2700Hz（機械的共振周波数）、50% デューティ  
**最大電流：** 3.3V / ~52.6Ω（実効インピーダンス）≈ 63mA（MLT-8530 定格範囲内）

### リレーコイルセンシング回路

リレーに並列接続された外部スイッチによる ON をセンシングする分圧回路。

```text
+12V ─── 100kΩ ─── Vsense ─── 27kΩ ─── GND
                      │
               ESP32 GPIO（入力）
```

| 状態 | Vsense | 論理 |
| ---- | ------ | ---- |
| リレーOFF（コイル断） | 12 × 27/(100+27) ≈ **2.55V** | HIGH（VIH = 2.475V ✓） |
| リレーON（Collector 引き込み） | ≈ 0V | LOW |

**負論理**（HIGH = リレー OFF）。コイル未接続時も 0V になる点に注意。

## 分圧回路

### 定数

| 定数 | 値 |
| ---- | -- |
| R1（上側） | 680kΩ 1% |
| R2（下側） | 22kΩ 1% |
| 分圧比 | 22k / (680k + 22k) ≒ 0.0313（1/31.9） |
| ADS1115 ゲイン | GAIN_EIGHT（±0.512V） |
| 理論分解能 | 約 1mV / LSB（バッテリー端換算） |

### 対応電圧範囲

- 最大: 約 16.4V（ADS1115 入力 0.512V = FSR）
- LiFePO4 4S: 10.0V（放電）～ 14.6V（充電）→ 0.313V ～ 0.457V
- 鉛バッテリー 12V: 10.5V ～ 14.7V → 0.329V ～ 0.460V

どちらも ±0.512V レンジ内に収まる。

## PCB レイアウト指針

### ゾーン分離

```text
┌──────────────────────────────────────────────────┐
│ [CN1: 12V入力]                                   │
│      ↓                                           │
│ F1 → TVS1 → D1                                  │
│                                                  │
│ ┌─[スイッチング領域]──┐    ┌─[アナログ領域]──────┐ │
│ │ LM2596             │    │ ADS1115            │ │
│ │ L1（スイッチングノード近傍）│ │ C7/C8（VDD直近）  │ │
│ │ D2                 │    │ R5/R6（プルアップ） │ │
│ │ C1 / C2            │    │                    │ │
│ └────────────────────┘    └────────────────────┘ │
│           ↑ GND は 1 点で結合                     │
│                                                  │
│ [CN2: BAT1] → R1/R2 → ADS1115 AIN0/1            │
│ [CN3: BAT2] → R3/R4 → ADS1115 AIN2/3            │
│                                                  │
│ [CN4-A: Grove → M5Atom S3]  ※案A               │
│ [ESP32-S3-MINI-1 + CN4-B: Grove → U128]  ※案B  │
└──────────────────────────────────────────────────┘
```

### 重要ルール

1. **LM2596 のスイッチングノード**（L1・D2 が接続されるノード）のパターンは短く・面積を小さく保つ。最もノイズを放射する箇所。
2. ADS1115 は LM2596 から**基板の反対側**に配置する。
3. 分圧抵抗（R1～R4）は ADS1115 の近くに置き、パターンを短くする。
4. C7/C8（デカップリング）は ADS1115 の VDD ピン直近に配置する。
5. GND プレーン（ベタ）を使う場合も、スイッチング領域とアナログ領域の GND を **1 点で接続**する（スター GND）。
6. 案B: ESP32-S3-MINI-1 の**全 GND ピンは基板 GND に接続**する（放熱・ノイズ対策）。

## 精度向上のための推奨事項

- 分圧抵抗は **1% 精度品**を使用（5% 品は不可）
- 同一温度ドリフト品の抵抗を近傍配置してペア動作させる
- 必要に応じてソフトウェアでキャリブレーション補正を追加する
- 高精度が必要な場合は変換レートを 8SPS に設定し複数サンプル平均化を行う

## USB-C 受電回路

PC や充電器から 5V を受電するための USB-C レセプタクル回路。

| Ref | 種別 | 仕様 | 備考 |
| --- | ---- | ---- | ---- |
| CN_USB | USB-C レセプタクル | USB 2.0 | SBU1/SBU2 は NC |
| R_CC1, R_CC2 | チップ抵抗 | 5.1kΩ | CC1/CC2 → GND プルダウン（UFP 識別） |
| D_ESD | ESD 保護 | LESD5D5.0CT1G（SOT-23） | VBUS-GND 間 TVS、D+/D− 保護内蔵 |
| C_ESD | セラミックコンデンサ | 100nF | TVS と並列（高周波 ESD バイパス） |
| D4 | ショットキーダイオード | SSL36 | OR ダイオード（12V 電源と USB-C の電源切替） |

```text
USB-C VBUS ─── D_ESD ─── C_ESD(並列) ─── D4(SSL36) ─── +5V バス
                                              ↑
                              LM2596 出力も同じノードに接続（OR 接続）

CC1 ─── 5.1kΩ ─── GND
CC2 ─── 5.1kΩ ─── GND
```

USB-C から給電時は D4 のアノード側が VBUS 電圧、カソード側が +5V バス。  
LM2596 が動作中は +5V > VBUS の場合 D4 が逆バイアスになり USB-C 側への逆流を防ぐ。

## 回路検証結果（m5atom_power_adc.kicad_sch）

### リレー駆動トランジスタ（Q3/Q4/Q7）

MMBT2222A、`(mirror y)` 配置。ピン位置を座標計算で検証済み。

- **E（pin2） → GND2**（コモンエミッタ確認）
- **C（pin3） → リレーコイル低電位側 + LED陰極**
- LED点灯電流：I = (12 - Vce_sat - Vf) / 5.1kΩ ≈ **1.9mA**（正常点灯）

> kicad-happy / kicad-netlist-tool の解析ツールは `mirror` + `angle` 組み合わせ時に
> E/C を誤判定するバグがある。実回路はコモンエミッタで正しい。

### RelayXSense 回路（R27-R32）

| 状態 | V_sense | 論理 |
| ---- | ------- | ---- |
| リレーOFF（コイル接続時） | 約 2.55V | HIGH |
| リレーON（Q または手動SW） | 約 0V | LOW |

**負論理**（HIGH=OFF）。コイル未接続時も 0V になる点に注意。

### USB-C SBU1/SBU2 ピン

USB 2.0 用途のみのため **NC（未接続）** が正しい。

## 拡張予定回路

### U128（SIM7080G）電源デカップリング

- U128 モジュール内部に 5V 側 22μF が搭載済み
- 外付けコンデンサ推奨値: **47μF + 100nF**（コネクタ端、X5R/X7R、16V 以上）
- 100nF は 50V 品でも問題なし

### サブバッテリー電流計測（INA228）

INA226 → LTC2944 と検討を経て、最終的に **INA228 を採用・実装済み**。

#### 部品比較

| 項目 | LTC2944 (C683672) | INA228 (C2862904) |
| ---- | ----------------- | ----------------- |
| パッケージ | DFN-8 (2×3mm) | VSSOP-10 |
| 自己電源 | SENSE+ から自己給電（3.6〜60V） | VS ピン必要（2.7〜5.5V） |
| 高圧側耐圧 | 60V | 85V |
| ADC | 16-bit | 20-bit |
| クーロンカウンタ | 16-bit | 40-bit |
| 電荷精度 | ±0.6% | ±1.0% FSR (max) |
| 電流精度 | ±0.4% | ±0.05% (max) |
| 温度測定 | あり（±3°C） | あり（±1°C） |
| I2C アドレス | 固定 0x64 | 16 通り（0x40〜0x4F） |
| LCSC 価格 | ~$12.4 | ~$3.0 |

**LTC2944 から INA228 への移行時の変更点:**

- フットプリント変更必須（DFN-8 → VSSOP-10）
- VS ピンに 3.3V 供給が必要
- I2C アドレス: 0x64 → 0x40（A0/A1=GND）※ 競合なし
- レジスタマップが異なる → ドライバ要書き直し

**INA228 注意事項（データシートより）:**

- 起動時に `SHUNT_CAL` レジスタを毎回書き込む必要あり（VS リセットで消える）
- ADCRANGE=1 使用時は `SHUNT_CAL` の計算値を×4する
- VS ピン直近に 100nF デカップリング必須
- IN+/IN- はケルビン接続推奨
- A0/A1 はフローティング禁止（GND/SCL/SDA/VS のいずれかに接続）
- VBUS は IN+ と同ノードに接続（PCB 上で直結可）

#### シャント抵抗選定

- バッテリー: LiFePO4 100Ah × 2 並列
- 配線: 14sq → 許容電流 約 80A（ボトルネック）
- シャント抵抗は**コネクタ経由で外付け**
- **ADCRANGE=1**（±40.96mV）採用
- 推奨シャント: **200A 75mV 品**（R = 0.375mΩ）
  - フルスケール電流: ±109A
  - 80A 時使用率: 73%
  - 電流 LSB: 208μA
- ログフォーマット: `x.xxA`（%.2f）
- 実用精度下限: 10mA 以上

### サブ → メインバッテリー緊急充電回路

メインバッテリー電圧低下時にサブから一時的に補充する回路。

**回路構成（AO3401A + MMBT2222A 流用）：**

```text
Sub(+) ─── AO3401A(P-ch) ─── Schottkyダイオード ─── R_limit ─── Main(+)
               ↑ MMBT2222Aレベルシフタ経由
           ESP32 GPIO
```

| 部品 | 値 | 備考 |
| ---- | -- | ---- |
| Q（Pch MOSFET） | AO3401A | 既存 BOM 流用 |
| レベルシフタ | MMBT2222A | 既存 BOM 流用 |
| R_limit | **4.7Ω 3W**（C3924519、2512 SMD） | TE Connectivity 35224R7JT |

**制御ロジック：**

```text
開始: V_main < 11.5V かつ V_sub > 13.0V
停止: V_main > 12.3V または 経過時間 > 20分（安全タイマー）
間欠動作: 電圧差（ΔV = V_sub − V_main）に応じて ON 時間を変化（すべて 60秒 ON で統一可）
```

最大電流: 2.5V / 4.7Ω ≈ **0.53A**（ΔV_max = 14V - 11.5V 想定）。
最大発熱: 0.53² × 4.7 ≈ **1.3W**（定格 3W に対して余裕率 2.3 倍）。

充電電流は測定しない。制御はすべて ADS1115 で読んだ電圧値（V_main / V_sub）のみで行う。
INA228 はサブバッテリーの全出力電流（残量積算）を監視するが、充電回路の制御には使わない。

> 充電電流の推定値は `I ≈ ΔV / 4.7Ω` で計算できる（センサー実測ではなく計算値）。

### LM2596 スイッチングノードの注意

スイッチングノード（LM2596 出力ピン〜D2〜L1 の三角ループ）はノイズのアンテナになるため、**ループ面積を最小化**することを最優先にする。トレース幅よりもパターンの短さ・小ささが重要。
